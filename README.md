# Banco_Java 💲
Este programa foi desenvolvido com base no "Curso mais didático e completo de Java e OO, UML, JDBC, JavaFX, Spring Boot, JPA, Hibernate, MySQL, MongoDB e muito mais" como parte de um exercício introdutorio a POO. 

Programa feito em Java, que faz uma demonstração de funcionamento de um banco. 

O primeiro passo é entrar com um número de conta, um nome do dono da conta e decidir se um depósito inicial será feito. 
Caso a resposta seja sim, o programa pede um valor de depósito. Caso não, ele segue para o próximo passo. 

Após a primeira etapa, ele irá mostrar os dados da conta. 

Em seguida, irá pedir um valor de depósito, que será acrescido a conta e irá mostrar os dados da conta atualizados. 

Por último, ele irá pedir um valor de saque (valor do saque + tarifa de R$5,00) e fará uma nova atualização dos dados. 

Tecnologias usadas: 
- Java
- Eclipse/SpringBoot
